HTML Block

Thiết kế > Blocks

Để tạo HTML block truy cập Admin > Thiết kế > Blocks và nhấn nút INSERT/Thêm, chọn tab HTML .

Và điền các trường sau đây:

Custom Block Name: tên block hiển thị trong Admin->Thiết kế->Bố cục
Custom Block Title: tên được hiển thị trên site
Use block wrapper (appearance): khuyến khích để mặc định
Use block's frame: bật để hiển thị khung giao diện block
Custom Block description: sử dụng nội bộ - không cần thiết
Custom Block content: sử dụng tích hợp sẵn trình soạn thảo HTML để thêm mã HTML. Ví dụ chèn YouTube video

Chú ý

Nhấn vào nút Source để chuyển chế độ soản thảo trực quan và dạng text . Mã HTML chỉ nên sửa ở chế độ in text.

Sau khi block đã được tạo nó cần được thêm vào trang mong muốn tại Admin > Thiết kế > Bố cục trước khi trạng thái này không thể bật.
