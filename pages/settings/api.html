Danh sách cài đặt API

Trang này có thể truy cập tại Hệ thống > Cài đặt > API

Storefront API status : Kích hoạt hoặc vô hiệu hóa store API.

Storefront API Key : Thiết lập chuỗi key để bảo mật hơn khi truy cập vào storefront API (Nhấn nút "generate/Tạo API"). Để trống nếu không sử dụng key.

Storefront API Stock Access : Kích hoạt nếu bạn cho phép storefront API để kiểm tra tình trạng còn hàng.

Admin API status : kích hoạt hoặc vô hiệu hóa Admin API.

Admin API Key : Thiết lập chuỗi key để bảo mật hơn khi truy cập vào Admin API.

Admin API Access IPs : Điền danh sách các IPs (ngăn cách dấu phẩy) sẽ được cho phép để truy cập admin API. Những IPs khác sẽ bị khóa.

Tasks API Key : Đây là khóa bí mật (secret key) cho phép để chạy task (scheduled processes) từ trình duyệt với file /task.php. Nếu không thiết lập, task sẽ không chạy qua trình duyệt.

Chú ý

Tasks API Key không được yêu cầu khi chạy task qua dòng lệnh (command line).

Hướng dẫn sử dụng API & bài tham khảo

Giới thiệu

API dạng RESTful dựa trên định dạng JSON. API hỗ trợ cả 2 giao thức HTTP & HTTPS cùng với cấu hình API key để bảo mật hơn.

API cung cấp tính năng tương tự với tính năng thông thường sử dụng trên trình duyệt và cung cấp nền tảng truy cập từ xa trên thiết bị di dộng để truy cập vào cửa hàng và thực hiện những thao tác thương mại điện tử.

API bao gồm sản phẩm và khách hàng. VD liệt kê danh mục, sản phẩm, thuộc tính sản phẩm cũng như tạo tài khoản khách hàng, đăng nhập, thêm sản phẩm vào giỏ và đặt hàng.

VinaCart API đã được tối ưu để thực hiện các thao tác một cách nhanh chóng.
API rt routes có địa chỉ dành riêng cho api, bắt đầu với a/... , VD, a/product/product


Kích hoạt API:
Mặc định API chưa được kích hoạt. Để kích hoạt API, bạn cần truy cập vào (admin) Cài đặt -> hệ thống và kích hoạt API. Nếu bạn muốn hạn chế truy cập với khóa bí mật, bạn có thể lưu nó trong cài đặt. Khóa này sẽ được yêu cầu cung cấp trên mỗi truy vấn API.

Chú ý

API hiện đang ở chế độ thử nghiệm. Hãy cẩn thận khi kích hoạt nó trên các trang web thực tế và sử dụng khóa API.
