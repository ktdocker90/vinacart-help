Danh sách menu chính

Design > Main Menu

Main Menu thường dùng cho menu ngang hiển thị bên dưới menu con ở góc phải trên cùng. Trang này hiển thị danh sách các liên kết đã tạo.

  • Để xóa một liên kết menu nhấn vào nút delete
  • Để một liên kết menu nhấn vào nút edit
  • Để thêm một liên kết menu sử dụng nút +.

Điền các trường sau đây:

- Unique ID - chữ & số không dấu cách;

- Text - tên link, được hiển thị trên website;

- Link - Liên kết đến trang có sẵn;

- Link data - Liên kết dữ liệu, link sẽ được sử dụng dữ liệu liên kết (hiếm dùng);

- Block ID - Tính năng này sẽ gắn và hiển thị block cho liên kết.

- Parent - chọn link là root hay hiển thị cấp con của link nào;

- Sort Order - thứ tự hiển thị bởi chỉ số.

- Icon - chọn hình ảnh đại diện cho link trên storefront. Chú ý: để hiển thị ảnh phụ thuộc vào template. ;

Công cụ hỗ trợ:

Link to Category

Link to Page

Các công cụ này sẽ tự động chèn chính xác category hoặc content page cho trường Link. Lựa chọn category từ danh sách drop-down "Link to Category" và nhấn vào nút LINK

Sau khi thay đổi link , nhấn nút `save` để lưu lại.
