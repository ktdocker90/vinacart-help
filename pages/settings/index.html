Quản lý khách hàng

Gồm các bài hướng dẫn giúp bạn học cách cài đặt và sử dụng nền tảng Vinacart

Cài đặt bằng tay

  • Settings/Cài đặt
    • Store details/Chi tiết cửa hàng
    • General/Tổng quát
    • Checkout/Thanh toán
    • Appearance/Giao diện
    • Mail
    • IM
    • API
    • System/Hệ thống

Các công cụ khác

  • Admin Users/Người dùng
  • Sao lưu và khôi phục
  • Quản lý tin nhắn
  • Data Export/Import (Nhập vào/xuất ra)
  • File Uploads (Tải lên)
  • Logs
  • Stock configuration (Tình trạng kho)
  • Encryption Data Manager (Quản lý bảo mật dữ liệu)
