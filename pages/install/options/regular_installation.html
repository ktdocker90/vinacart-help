Tải gói mới nhất của Vinacart.

Tải VinaCart

Tạo một thư mục chứa toàn bộ mã nguồn. Sau khi tải về, tiến hành giải nén (unzip), File nén sẽ bung toàn bộ các files và thư mục sử dụng để chạy vinacart.

Upload

Nếu cài đặt trên hosting bạn sẽ cần upload tất cả các files và thư mục được yêu cầu này lên web server của bạn

Yêu cầu phần mềm FTP

Tải lên toàn bộ các file trên server bằng phần mềm FTP, VD. FileZilla.

Thiết lập quyền cho files

Bạn cần phân quyền chính xác cho tệp và thư mục.

Phần mềm FTP

Nếu bạn không có quyền truy cập SSH vào máy chủ của bạn, bạn có thể thay đổi quyền truy cập tập tin bằng cách sử dụng phần mềm FTP. Hầu hết chúng đều có tùy chọn "Change permissions" hoặc chức năng "Change mode". Với sự trợ giúp này bạn có thể thiết lập phân quyền cho file 'config.php' và các thư mục được liệt kê ở trên. Lệnh 'chmod 666 config.php' có nghĩa quyền hạn 666 (quyền đọc, viết và thực thi cho người dùng sử dụng tệp , nhóm ..) cho file 'config.php'. Ví dụ khác ('chmod -R 777 catalog') quyền 777 tương tự cho thư mục, các thư mục con & toàn bộ files trong thư mục.

Phần quyền tự động

Bạn có thể phân quyền cho các files và thư mục tự động bằng cách chạy file caidat.php hoặc có thể kiểm tra lại quyền ở dưới đây để đảm bảo cài đặt vinacart được hoạt động đúng.

Linux/Unix server hãy đảm bảo thư mục và các tệp sau có thể viết.

copy 
chmod 0777 system/config.php
chmod 0777 system/
chmod 0777 system/cache/
chmod 0777 system/logs/
chmod 0777 image/
chmod 0777 image/thumbnails/
chmod 0777 download/
chmod 0777 extensions/
chmod 0777 admin/system/backup/ (if exist)
chmod 0777 resources/

Windows server hãy đảm bảo thư mục & files cho phép đọc và viết.

copy 
system/config.php
system/
system/cache/
system/logs/
image/
image/data/
image/thumbnails/
download/
resources/

Một một CSDL MySQL

Tạo một cơ sở dữ liệu MySQL thông qua web control panel. Ghi nhớ tên database MySQL , host, user, và mật khẩu; thông tin này sẽ được yêu cầu trong quá trình cài đặt.

Trình cài đặt tự động

Mở trình duyệt vào chạy vào file cài đặt.

Nếu bạn đặt files VinaCart trực tiếp tại web root, mở http://www.yourdomainname.com/

Nếu bạn đặt files VinaCart vào thư mục con giống như web_root/store/, mở http://www.yourdomainname.com/store/

    Đồng ý chính sách

    Vui lòng đọc và chấp thuận điều lệ sử dụng. Nếu bạn sẵn sàng cài đặt và đồng ý những điều lệ của vinacart, thì chọn vào 'Tôi đồng ý' và nhấn nút tiếp tục.
  • 2

    Thông tin cấu hình server

    Phần này để bạn kiểm tra cấu hình PHP, MySQL, phần mở rộng được yêu cầu như PHP CURL, ZIP & GD trên server và quyền viết. Nếu thỏa mãn , bạn sẽ nhấn nút tiếp tục. Nếu không hãy liên hệ với nhà quản trị server của bạn hoặc hosting.
  • 3

    Cấu hình thông tin (Host & Database)

    Vui lòng điền chi tiết thông tin của web server host và MySQL server cũng như thông tin khác. Điền đầy đủ vào các trường sau đây:

    Nhập thông tin CSDL

    Database Driver: AbanteCart uses a MySQL database but you might have to use an MySQLi driver
    Hostname: thường tên là 'localhost', nếu cài trên máy hoặc một tên host khác nếu sử dụng database ngoài
    Database Username: tên người dùng CSDL MySQL
    Database Password: Mật khẩu người dùng MySQL
    Database Name: Tên CSDL MySQL
    Nhập tên bí mật sử dụng để đăng nhập vào trang quản trị. Chú ý: Đừng chia sẻ chuỗi này cho bất kỳ ai.
    Chuỗi bí mật (unique key): sẽ sử dụng để đăng nhập vào quản trị: http://www.yourdomainname.com/index.php?s=your_admin .

    Nhập thông tin tài khoản người dùng và cấu hình site.
    Người dùng admin: tên đăng nhập cho admin ;
    Mật khẩu admin: mật khẩu admin;
    Xác nhận mật khẩu: xác nhận mật khẩu admin;
    Admin E-Mail: Nhập một e-mail thưc vào trường này. Đây là e-mail cho người quản trị, sử dụng trong các trường hợp như đặt hàng, email nhận từ form liên hệ.
    Tên cửa hàng: nhập tên cửa hàng.
    Giao diện: Chọn giao diện cài đặt mặc định.

    Nạp dữ liệu

    Nhấn nút cài đặt để bắt đầu cấu hình và cài dữ liệu mẫu gồm sản phẩm, đơn hàng,danh mục, ...
  • 5

    Hoàn thành

    Sau khi cài đặt đã hoàn tất, nhớ mật khẩu và tên đăng nhập vào quản trị và keep secure key save. Xóa thư mục install và thiết lập quyền 444 cho file system/config.php và /index.php
