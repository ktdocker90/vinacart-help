Hiểu về Resource Library (RL)

Hệ thống trang bị linh hoạt và thuận tiện để lưu trữ, quản lý và gán tài nguyên Files.

Hệ thống có thể xử lý các tập tin media hoặc bất kỳ tập tin khác hoặc mã HTML gán cho sản phẩm, danh mục, thương hiệu hoặc bất kỳ nguồn dữ liệu khác. Ví dụ đơn giản cho resource là hình ảnh có thể gán cho một hoặc nhiều sản phẩm hoặc danh mục. Ngoài ra bạn có thể sử dụng Resource Library để thêm ảnh cho sản phẩm hoặc nội dung banner.
Resource library có thể sử lý tải files hoặc nhập mã HTML/Javascript . Các files được tải lên tự động sử lý tạo ra hình thu nhỏ (thumbnails).

Truy cập Resource Library

Media hoặc Resource Library có thể truy cập từ mọi sản phẩm, danh mục etc. Bạn cũng có thể truy cập từ Admin > Catalog > Media Manager. Trên trang này bạn có thể thêm/sửa kiểu file cho phép download và thư mục nơi file tải lên.

Tải lên sử dụng Resource Library (RL)

  • 1

    Tải nhiều hình ảnh hoặc một ảnh bằng cách kéo thả Drag-n-Drop

    Kéo một ảnh từ máy tính của bạn vào vùng thả để tải tự động files lên server và gán nó vào các sản phẩm, danh mục, thương hiệu.. vv.
    Lưu ý về hỗ trợ trình duyệt: Microsoft Edge có một lỗi với tính năng kéo thả.
  • 2

    Tải files sử dụng nút Add Resource

    Đôi khi cách kéo và thả files không phải là cách thuận tiện để tải file lên server. Đặc biệt khi người dùng sử dụng máy tính màn hình nhỏ. Do đó trong trường hợp này họ sẽ chọn cách phù hợp nhất. Người dùng có thể tải file sử dụng nút Upload.
  • 3

    upload nhiều bản sao từ cùng một file

    Nếu bạn muốn gán 'Map (Link)' cho sản phẩm cho một file media có sẵn chỉ việc nhấn vào tab "Resource library" khi bạn sửa sản phẩm trên cửa sổ RL . Ở đây bạn có thể điều hướng hoặc tìm kiếm cho bất kỳ file nào đã được tải lên.

Thêm Images với HTML như một tài nguyên

Images cần được chèn trong thẻ HTML (không chỉ URL):
<img alt="Ideal Open Source eCommerce Solution" src="http://www.example.com/images/logo.png">

Thêm mã ảnh gán cho sản phẩm của bạn sử dụng Resource Library trong tab Media.

Thêm YouTube video như một tài nguyên cho sản phẩm

vào trang YouTube và lấy mã nhúng video. YouTube tạo nó cho bạn trong tab Share :
<iframe src="http://www.youtube.com/embed/AhgtoQIfuQ4" frameborder="0" width="640" height="360"> </iframe>

Tiếp đó, thêm mã nhúng video vào sản phẩm sử dụng Resource Library trong tab Media . Hãy chắc chắn chọn 'video' resource type, xem hình

Bạn cần tạo một Data Listing Block với Media và kích hoạt nó trong bố cục trang sản phẩm mặc định layout.

* Để tạo block truy cập Admin > Design > Blocks. Sử dụng loại Data Listing Block . Hãy đảm bảo chọn Listing Data Source: Media và Media Type: Video

Sau đó truy cập Admin-Design-Layout chọn bố cục "Default Product Page" và kích hoạt block video

Sau khi đã lưu, truy cập vào trang sản phẩm trên web để xem video
