Hãy xem cách tạo Data Listing: Media Audio Block

Data Listing block

Để tạo Data Listing block truy cập Admin > Thiết kế > Blocks

Trong Admin > Thiết kế > Blocks và nhấn vào nút INSERT , chọn tab Data Listing . Sau đó điền những trường sau:
Custom Block Name: tên block hiển thị trong Admin > Thiết kế > Bố cục
Custom Block Title: tên được hiển thị trên web
Use block wrapper (appearance): khuyến khích để mặc định.
Use block's frame: bật để hiển thị khung giao diện block.
Custom Block description: sử dụng nội bộ - không cần thiết
Listing Data Source: chọn nguồn liệt kê dữ liệu:

  • Popular Products
  • Special Offers
  • Categories
  • Manufacturers
  • Featured Products
  • Latest Products
  • Bestsellers Products
  • Media
  • Custom Products
  • Custom Categories
  • Custom Manufacturers

Chú ý

Sau khi bạn chọn nguồn liệt kê dữ liệu các cài đặt thêm sẽ được hiển thị:
trường Limit cho một số nguồn dữ liệu chung: Popular, Special, Categories
thêm/sửa đối với custom types như Custom Products, Custom Categories và Custom Manufacturers
Media Type cho nguồn Media

Sau khi block được tạo nó cần được thêm vào page bạn mong muốn trong Admin > Thiết kế > Bố cục. Trước khi tính năng này không thể bật.
