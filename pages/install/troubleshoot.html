  • Cảnh báo: Nếu server của bạn có kích hoạt opcache php module. Vui lòng tắt đi trước khi tiến hành cài đặt!

    Giải pháp: Thực hiện một trong hai cách sau để tắt opcache:
    1. Nếu PHP chạy dưới Apache module, sử dụng file .htaccess:
    php_flag opcache.enable Off

    2.Nếu PHP chạy dưới CGI/FastCGI, sử dụng file .user.ini:
    opcache.enable=0
