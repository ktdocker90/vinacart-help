Cài đặt Blog

Cài đặt chi tiết

Phần này thiết lập cấu hình chung cho blog.

Blog Title - Hiển thị trên thanh tiêu đề khi truy cập vào trang blog (e.g: /blog)

Description - Mô tả blog

Top Message - Xuất hiện ở phía trên trang blog chính.

Bottom Message - Xuất hiện ở phía dưới trang blog chính.

Meta Tag Description - Thẻ meta cho description nằm trong phần head.

Meta Keywords - Thẻ meta cho keywords nằm trong phần head.
