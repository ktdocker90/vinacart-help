AbanteCart System Requirements

AbanteCart runs on most server configurations, Windows or Linux based servers that utilize different PHP configurations.

Bắt đầu với VinaCart

PHP version
Yêu cầu PHP v 5.4 hoặc cao hơn. Cấu hình PHP sau được yêu cầu:
copy 

	Register Globals: Off
	Magic Quotes GPC: Off
	File Uploads: On
	Session Auto Start: Off

MySQL

database
MySQLi database phiên bản 5.0+ hoặc cao hơn, MySQL, PDO_MYSQL.

cURL

PHP extension
PHP CURL là cần thiết cho hoạt động của một số cổng thanh toán và hệ thống (vd PayPal, Authorize.NET etc) và dịch vụ vận chuyển thời gian thực (FedEx, UPS).

GD

library
Thư viện GD php cần được cài đặt

ZIP

extension
ZipArchive phần mở rộng này yêu cầu kích hoạt.

Pcntl

extension
Không bắt buộc, sử dụng để chạy background job cho các tác vụ sử lý lâu (như gửi email khi đặt hàng...).

Redis

extension
Không bắt buộc kích hoạt cùng với Pcntl extension.

ionCube

extension
Yêu cầu cài đặt ioncube loader phiên bản 5.6 trở lên.

Vinacart sẽ không làm việc chính xác hoặc không hoạt động nếu không thỏa mãn cấu hình trên.
