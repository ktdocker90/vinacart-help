Cài đặt trên localhost

Để cài đặt Vinacart trên localhost, yêu cầu đã cài bộ web server chạy php như (XAMPP, WAMPP,..). There are 3 most popular third party applications available that provide virtual layer with web server and database functionality. Chú ý: XAMPP hỗ trợ trên máy MAC, Window và Linux.

XAMPP

XAMPP là phần mềm hoàn toàn miễn phí, dễ dàng cài đặt Apache bao gồm MySQL, PHP, và Perl.

XAMPP

http://www.apachefriends.org/en/xampp.html

Tài liệu http://www.apachefriends.org/en/faq-xampp.html
Sau khi XAMPP được cài đặt và cấu hình:

  • Giải nén các files & thư mục từ file nén tới đường dẫn xampp_install_path\xampp\htdocs\vinacart_folder_name\
  • Tạo mới database.
  • Nếu cài đặt trên localhost, sử dụng xampp, wamp.. bạn cần tạo alias (vd: vinacart.dev) thay vì sử dụng tên localhost. Ví dụ: Edit file c:/xampp/conf/extra/httpd-vhosts.conf 
        <VirtualHost *:80>
        ServerName vinacart.dev
        ServerAlias www.vinacart.dev
        DocumentRoot "C:\xampp\htdocs\vinacart"
        <Directory "C:\xampp\htdocs\vinacart">
            Order allow,deny
            Allow from all
        </Directory>
            ErrorLog C:\xampp\htdocs\error.txt
    </VirtualHost>
  • Thiết lập quyền truy cập Files
    .

    Nếu chạy web trên hệ điều hành Window + xampp/wamp thì bỏ qua bước này. Trường hợp sử dụng Linux, để thiết lập chmod đầy đủ cho các file chúng ta chạy file caidat.php trên trình duyệt: http://vinacart.dev/caidat.php

  • Bắt đầu cài đặt vinacart: Mở trình duyệt, truy cập trang cài đặt: http://vinacart.dev/install/ Làm theo trình tự các bước (wizard).

    Trong quá trình cài đặt sẽ hỏi thông tin sau :
    Database Host: localhost
    User: root
    Password: để trống (mặc định với xampp)
    Database Name: tên database vừa tạo.

    Chú ý: nếu bạn muốn cài đặt lại vinacart, sử dụng URL sau: http://vinacart.dev/install/?rt=install&force=1
